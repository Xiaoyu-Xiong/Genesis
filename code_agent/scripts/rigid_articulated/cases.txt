diff_drive_waypoints|Create a differential-drive mobile robot with two independently actuated wheels and three visible floor waypoint markers. Over 8s, use a simple closed-loop controller that reads the robot base pose each step, steers from heading and distance error, slows near each waypoint, and advances to the next waypoint only after reaching a distance threshold.
forklift_pallet_align|Create a primitive forklift-like articulated vehicle, a pallet-sized box, and a marked loading zone. Over 9s, use closed-loop steering based on measured lateral and heading error to align the vehicle with the pallet, drive the forks under it, raise the fork mast to a target height using lift-height feedback, and then reverse slightly while holding the pallet level.
scara_target_servo|Create a fixed-base SCARA-style robot with two revolute arm joints, one vertical prismatic joint, and four colored target spheres on a tabletop. Over 8s, use closed-loop joint or end-effector feedback to servo the tool tip to each target in sequence, switching targets only when the end effector is within a small distance threshold.
pan_tilt_tracker|Create a fixed pan-tilt turret with yaw and pitch actuators, a visible pointer beam, and a bright ball moving along a simple scripted arc. Over 8s, use closed-loop yaw and pitch control from the measured vector and angular error between turret and ball, keeping the pointer within a small angular target threshold without directly setting joint positions mid-simulation.
gripper_cube_lift|Create a fixed-base 3-DOF pick arm with a two-finger gripper, a movable cube, and a raised target platform. Over 9s, use a staged closed-loop controller: approach the cube from end-effector position error, close the gripper until finger gap/contact proxy reaches a threshold, lift to a target height using height feedback, and carry the cube toward the platform.
box_pusher_goal|Create a small articulated pusher robot with steerable wheels, a front bumper, one movable box, and a floor goal zone. Over 8s, use closed-loop control from measured robot and box positions to first align the bumper behind the box, then push while correcting heading so the box moves into the goal zone.
door_arm_servo|Create a fixed two-link arm beside a lightweight hinged door panel with a visible handle. Over 8s, use closed-loop end-effector control from measured hand-to-handle error to move the hand to the handle region, apply actuator/force-based pushing while tracking measured door angle, and stop once the hinge reaches a target open-angle threshold.
gantry_pick_place|Create an overhead gantry robot with actuated X, Y, and Z prismatic axes plus a simple gripper, one movable puck, and two bin markers. Over 9s, use closed-loop axis control from measured end-effector and puck positions to center above the puck within a small XY threshold, lower until a height/contact threshold, grip, lift, move to the selected bin, and release.
balance_and_drive|Create a two-wheel self-balancing robot with an upright body link and a floor goal marker. Over 8s, use a simple PD-style closed-loop controller from body tilt, angular velocity, and base position error to keep the robot upright while it slowly drives toward the marker; keep the motion modest and visually stable.
dual_arm_handoff|Create two fixed-base articulated arms facing each other with a small baton-shaped rigid object between them. Over 9s, use simple closed-loop end-effector controllers from measured hand, baton, and handoff-point errors so the first arm approaches and presents the baton, the second arm tracks the same point, closes its gripper once within a distance threshold, and then lifts the baton slightly.
